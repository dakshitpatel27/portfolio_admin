Input the deployed Web App URL from your Google Apps Script. This connects your portfolio dashboard and main site directly to the JSON file on your Google Drive.

Set your Admin Panel password here. When you click **Save Configuration** (sticky bar at bottom), it will update `config.js` with your new password.

Setup Instructions:

  1. Create a new script at script.google.com.
  2. Copy and paste the code from google-apps-script.js (located in your project root) into the script editor.
  3. Click Deploy > New deployment.
  4. Select type Web app. Configure:
    • Execute as: Me
    • Who has access: Anyone
  5. Click Deploy, authorize permissions, and copy the generated Web App URL.
  6. Paste it in the field above!
